## About

Python and Rust engineer based in **Thailand**, focused on the intersection of financial systems, data engineering, and applied machine learning.

I build tools that analysts and engineers actually rely on — investment analyzers, production data pipelines, and anomaly detection systems. Alongside applied work, I've been systematically deepening **Rust** proficiency: from building a text editor and screen recorder from scratch to completing Rustlings, Advent of Code, and a full backend course.

> *Python for domain logic and rapid iteration — Rust when correctness and speed both matter.*
